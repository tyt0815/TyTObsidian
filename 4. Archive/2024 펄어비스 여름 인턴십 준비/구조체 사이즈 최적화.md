C/C++에서 구조체의 사이즈는 멤버의 가장 큰 데이터 타입에 따라 정렬된다.
```cpp
struct Test
{
	char c;
};
```
Size: 1
```cpp
struct Test
{
	short s;
};
```
Size: 2
```cpp
struct Test
{
	int i;
};
```
Size: 4
```cpp
struct Test
{
	double d;
};
```
Size: 8

```cpp
struct Test
{
	char c;
	short s;
};
```
Size: 4
이경우 2바이트 경계 정렬 이기 때문에 c가 1바이트 + 1바이트 패딩, s가 2바이트로 총 4바이트가 된다.
```cpp
struct Test
{
	char c;
	int i;
};
```

```cpp
struct Test
{
	int i;
	char c;
};
```
Size: 8
이경우 4바이트 경계 정렬 이기 때문에 c가 1바이트 + 3바이트, i가 4바이트로 총 8바이트가 된다.

```cpp
struct Test
{
	int i1;
	int i2;
	char c1;
	char c2;
};
// Size : 12
```

```cpp
struct Test
{
	int i1;
	char c2;
	int i2;
	char c2;
};
// Size : 16
```
위 경우가 정렬이 되지 않아 구조체 사이즈가 늘어난 경우이다.
