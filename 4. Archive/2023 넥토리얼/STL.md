# STL(Standard Template Library)란?
표준 템플릿 라이브러리는 C++용 일반 프로그래밍 도구 세트를 제공하는 라이브러리
다음으로 이루어져 있음
1. 컨테이너
2. 알고리즘
3. 이터레이터
4. 함수 객체
## 컨테이너
컨테이너는 데이터를 저장하고 구성하는 방법을 제공하는 클래스
## 알고리즘
알고리즘은 컨테이너에서 작동하여 정렬, 검색, 복사 등을 수행하는 함수
## 이터레이터
이터레이터는 컨테이너 요소에 액세스할 수 있는 방법을 제공하는 객체
## 함수 객체
함수처럼 동작하는 객체
